Class AppFor

Namespace
BOOSEapp

BOOSEapp.dll

Represents a "for" loop command that iterates a variable form a starting value to an end value with an optional step icrement. The command supports syntax such as "for i = 1 to 10 step 2", where "i" is the loop variable, "1" is the starting value, "10" is the ending value, and "2" is the step.
